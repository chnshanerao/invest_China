## 一、项目背景

### 1.1 解决什么问题

A股散户面临的核心困境：
- **追涨杀跌**：凭感觉交易，没有系统化的入场/出场规则
- **不会止损**：利润坐过山车，赚30%最后亏着出来
- **盯盘焦虑**：没有自动化的信号监测和提醒

### 1.2 系统定位

一套**纯右侧趋势跟踪系统**，覆盖36个A股行业/策略ETF：
- **入场**：4条件全满足才买（趋势确认，不抄底）
- **出场**：自适应ATR追踪止损（涨得越多止损越紧，自动锁利润）
- **全自动监测**：每日扫描信号 → 推送钉钉 → 无需盯盘

### 1.3 原型验证结果（500天回测，36个ETF）

| 指标 | 数值 |
|------|------|
| 平均收益 | +47.1%（同期上证+26.3%） |
| 盈利比例 | 26/36（72%的ETF盈利） |
| 头部标的 | 芯片+178%, 通信+155%, 半导体+153% |
| 总交易 | 355笔，胜率48% |
| 核心优势 | 大赚小亏——盈利单平均远大于亏损单 |

## 三、核心交易逻辑

### 3.1 设计哲学

A股交易不看PE估值。A股5000+标的只有约500家年利润超10亿，PE轻松上百上千。
**看趋势、看资金、看动量、看政策、看概念。**

系统核心思路：
- 趋势来了上车，趋势走了下车
- 用数学方法（ATR自适应）代替人的情绪判断
- 大赚小亏：让利润奔跑，快速砍掉亏损

### 3.2 右侧入场信号

**4个条件全部满足才触发入场**（缺一不可）：

| # | 条件 | 技术实现 | 含义 |
|---|------|---------|------|
| 1 | 价格 > MA20 | `close > SMA(close, 20)` | 站上中期均线 |
| 2 | MA20上行 | `SMA(close, 20)[today] > SMA(close, 20)[5日前]` | 均线拐头确认 |
| 3 | MACD柱 > 0 | `MACD_histogram > 0` | 动量为正 |
| 4 | 量价确认 | `5日均量 > 20日均量` 或 `量比>1.2 且 当日上涨` | 资金流入 |

**可选加分项**（不强制，用于展示）：
- 创20日新高
- 价格 > MA60（站上长期均线）
- RSI > 50（强势区域）
- MACD金叉

**前端展示**：每个条件显示 ✅/❌，让用户一眼看到差哪个条件。

### 3.3 自适应ATR追踪止损（核心创新）

这是系统最核心的部分。传统止损用固定百分比（如-8%），问题是：
- 涨了30%还用-8%止损？利润全回吐
- 暴涨阶段波动大，-8%可能被正常波动洗掉

**解决方案**：基于ATR（真实波幅均值）的动态追踪止损，止损系数K随盈利幅度和加速度自适应调整。

#### 3.3.1 公式

```
trailing_stop = highest_close_since_entry - K × ATR(20)

K = max(1.2, 3.0 - gain_adj - accel_adj)

其中:
  gain_adj  = min(累计涨幅% / 10, 6) × 0.3    // 每涨10%，K收紧0.3
  accel_adj = 0.3 if 近5日涨幅 > 8% else 0     // 急涨额外收紧
```

#### 3.3.2 K值对照表

| 累计涨幅 | 近5日 ≤ 8% | 近5日 > 8%（急涨） |
|---------|-----------|-----------------|
| 0% | K=3.0（宽松） | K=2.7 |
| 10% | K=2.7 | K=2.4 |
| 20% | K=2.4 | K=2.1 |
| 30% | K=2.1 | K=1.8 |
| 40% | K=1.8 | K=1.5 |
| 50%+ | K=1.5 | K=1.2（最紧） |

**直觉解释**：
- 刚入场时给3倍ATR的空间，让趋势发展
- 涨得越多，止损越紧，锁住利润
- 暴涨加速时额外收紧，防止急跌吃掉利润
- 最低K=1.2，保证不会被正常波动洗出

#### 3.3.3 硬止损兜底

无论ATR止损怎么算，永远有一个硬止损：

```
hard_stop = entry_price × 0.92    // 最大亏损8%
```

#### 3.3.4 实战案例（回测数据）

半导体设备ETF(sz159516)，7笔交易全部盈利：

| 入场 → 出场 | 收益 | 出场K值 | 说明 |
|-------------|------|---------|------|
| 2024-09-27 → 10-10 | +22.2% | K=1.7 | 涨多了K自动收紧 |
| 2025-06-24 → 09-04 | +11.0% | K=2.3 | 中等涨幅中等K |
| 2026-04-14 → 05-29 | +33.9% | K=1.5 | 大涨后K极紧，锁住利润 |

### 3.4 出场规则

**任一条件触发即出场**：

| 优先级 | 出场条件 | 实现 |
|--------|---------|------|
| 1 | 追踪止损 | `close < highest_close - K × ATR(20)` |
| 2 | 硬止损 | `close < entry_price × 0.92` |
| 3 | 趋势破坏 | 连续3个交易日收盘价 < MA20 |

### 3.5 仓位管理规则

| 规则 | 值 | 说明 |
|------|---|------|
| 最大同时持仓 | 5个 | 集中但分散 |
| 单个仓位 | 15-20%资金 | 等权分配 |
| 入场资金 | 一次性全仓 | 信号确认即全额买入 |
| 回测弱标的 | 自动跳过 | 500天回测收益<10%的不提示信号 |

### 4.3 技术指标计算

所有指标为纯Python实现，无第三方依赖：

| 指标 | 函数 | 参数 | 用途 |
|------|------|------|------|
| SMA | `sma(data, period)` | period=5/10/20/60 | 均线 |
| EMA | `ema(data, period)` | period=12/26 | MACD组件 |
| RSI | `calc_rsi(closes, 14)` | period=14 | 强弱指标 |
| MACD | `calc_macd(closes)` | 12/26/9 | 动量+金叉 |
| Bollinger | `calc_bollinger(closes, 20, 2)` | 20日2倍标准差 | 通道 |
| ATR | `calc_atr(highs, lows, closes, 20)` | period=20 | 波幅→止损 |
| 量比 | `calc_volume_ratio(volumes, 20)` | period=20 | 量能 |

- **标的管理**：添加/删除/编辑ETF_BASKET，修改分类
