# A股ETF右侧趋势交易系统

基于自适应ATR追踪止损的右侧趋势跟踪系统，覆盖36个A股行业/策略ETF。纯Python实现，零第三方依赖。

## 核心理念

- **只做右侧** — 趋势确认后入场，不抄底不猜底
- **自适应止损** — 涨得越多止损越紧，自动锁住利润
- **大赚小亏** — 让利润奔跑，快速砍掉亏损
- **全自动监测** — 每日扫描信号，钉钉推送，无需盯盘

## 交易逻辑

### 入场信号（4条件全满足）

| # | 条件 | 实现 | 含义 |
|---|------|------|------|
| 1 | 价格 > MA20 | `close > SMA(close, 20)` | 站上中期均线 |
| 2 | MA20上行 | `MA20(今) > MA20(5日前)` | 均线拐头确认 |
| 3 | MACD柱 > 0 | `MACD_histogram > 0` | 动量为正 |
| 4 | 量价确认 | `5日均量 > 20日均量` 或 `量比>1.2且当日上涨` | 资金流入 |

### 自适应ATR追踪止损（核心）

```
trailing_stop = highest_close_since_entry - K × ATR(20)

K = max(1.2, 3.0 - gain_adj - accel_adj)
  gain_adj  = min(累计涨幅% / 10, 6) × 0.3    # 每涨10%，K收紧0.3
  accel_adj = 0.3 if 近5日涨幅 > 8% else 0     # 急涨额外收紧
```

| 累计涨幅 | 正常K | 急涨K |
|---------|-------|-------|
| 0% | 3.0（宽松） | 2.7 |
| 10% | 2.7 | 2.4 |
| 20% | 2.4 | 2.1 |
| 30% | 2.1 | 1.8 |
| 50%+ | 1.5 | 1.2（最紧） |

刚入场给3倍ATR空间让趋势发展，涨得越多止损越紧锁住利润，暴涨时额外收紧防急跌。

### 出场规则（任一触发）

1. **追踪止损** — `收盘价 < highest_close - K × ATR(20)`
2. **硬止损** — `收盘价 < 入场价 × 0.92`（最大亏损8%）
3. **趋势破坏** — 连续3个交易日收盘价 < MA20
